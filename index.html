Tunnel

Chrome Extensions <--> Web Pages

What is Tunnel?

Chrome extensions are an interesting beast. They're awesome because.. well.. they let you inject your code into other people's code! So obviously there are a lot of security concerns here. That's why when your injected code runs, it cannot modify the globals of the page. The only thing it shares is the DOM. This isn't that big of a deal. But as pages are becoming more dynamic, we're losing the ability to make assumptions about the state of the DOM.

To fix this, Tunnel provides a messaging system that allows the page and the Chrome extension to have two way communication. This means you can inject code into the runtime that'll notify the extension when a certain event occurs. Effictively bringing back synchronized access to the DOM.
